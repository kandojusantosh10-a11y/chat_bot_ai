
📌 Project Title

AI Chatbot using Python & Streamlit

📝 Description

This project is a simple AI-powered chatbot built using Python and Streamlit. It provides an interactive chat interface where users can communicate with the bot in real time. The chatbot can respond to basic queries such as greetings, Python-related questions, and project suggestions.

The application includes a clean user interface, chat history memory, and a sidebar for managing settings like clearing conversations. It is designed as a beginner-to-intermediate level project to demonstrate fundamental concepts of chatbot development and UI design using Streamlit.

🚀 Features
💬 Interactive chat interface
🧠 Basic intelligent responses (rule-based logic)
🗂️ Chat history memory
⚙️ Sidebar with clear chat option
🎨 Clean and responsive UI

🛠️ Tech Stack
Python
Streamlit

▶️ How to Run
pip install streamlit
streamlit run app.py
